Introducción a HTML y HTML5

1. ¿Qué es HTML?

HTML, sigla en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto), es el lenguaje de marcado estándar para la creación de páginas web. Define la estructura y el contenido de una página web usando un código llamado código HTML.

2. ¿Qué es HTML5?

HTML5 es la quinta y última versión del lenguaje HTML. Introduce nuevas características y mejoras sobre HTML4.01, como un mejor soporte para multimedia, nuevos elementos semánticos y una mayor interoperabilidad con tecnologías modernas. HTML5 incluye elementos como <header>, <footer>, <article>, y <section> que permiten una estructura más clara y significativa del contenido.

3. Elementos Básicos de Texto

HTML permite estructurar contenido utilizando títulos, párrafos y otros elementos de texto. Los títulos se definen mediante las etiquetas de encabezado, como <h1>, <h2>, etc., mientras que los párrafos se crean con la etiqueta <p>.

Para insertar un salto de línea simple, se utiliza la etiqueta <br>. Esta etiqueta es útil para separar líneas dentro de un párrafo.
